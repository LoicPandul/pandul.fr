**Step 1: Font Selection (per D-05, D-06, D-07, D-08)**

The user explicitly decided to CHANGE BOTH fonts. The current JetBrains Mono + Manrope pair was judged "incoherent". The UI-SPEC kept them -- OVERRIDE the UI-SPEC on this point per user decision D-05.

Requirements:
- HEADINGS font: geometric / modern style (D-06) -- think clean, technical, contemporary. NOT monospace.
- BODY font: classic, highly readable for long-form content (D-07), well-paired with the heading font
- Both must be modern and coherent together (D-08)
- Must be available as variable fonts (or at minimum weights 400 + 700)
- Must be freely licensed (Google Fonts or similar)
- Must be downloaded as local files (not CDN -- per project constraint of no external dependencies)

- Headings: a geometric sans like Sora, Outfit, Satoshi, Instrument Sans, General Sans, Plus Jakarta Sans, or similar contemporary geometric faces
- Body: a humanist/transitional serif or clean sans like Source Serif 4, Libre Baskerville, Literata, or a complementary sans

Download the chosen fonts as TTF or WOFF2 variable files to `assets/fonts/`. Archive the old fonts by moving JetBrainsMono-Variable.ttf and Manrope-Variable.ttf to `assets/fonts/archive/`.

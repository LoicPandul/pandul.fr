  <name>Task 1: Visual and functional verification of complete dictionary</name>
  <what-built>
Complete dictionary browsing experience with:
- 1408 enriched definition pages (breadcrumbs, category badge, English term, cross-ref pills, prev/next nav, book CTA, image lightbox)
- Real-time search autocomplete on landing + definition pages
- A-Z letter navigation (26 pages)
- 19 category pages with local A-Z filtering
- Dictionary landing page with stats hero, search, book promo, A-Z bar, category grid
  </what-built>
  <how-to-verify>
Start local Jekyll server: `bundle exec jekyll serve`

**Landing page (http://localhost:4000/dictionnaire/):**
1. Verify H1 "Dictionnaire de Bitcoin" and stats "1408 definitions dans 19 categories"
2. Type "utxo" in search bar — verify dropdown shows matching definitions
3. Type "eclair" (no accent) — verify accent-insensitive match for ECLAIR entries
4. Use keyboard: Arrow Down to navigate results, Enter to select
5. Verify book promo section is intact (cover image, Amazon + Bitcoin Bazar buttons)
6. Verify A-Z bar with 26 letter buttons
7. Click letter "B" — verify it goes to /dictionnaire/lettre/b/
8. Verify 19 category cards with correct counts
9. Click "Protocole" card — verify it goes to category page

**Category page (e.g., /dictionnaire/categorie/protocole/):**
10. Verify breadcrumbs: Dictionnaire > Protocole
11. Verify "168 definitions" count
12. Verify A-Z bar with local anchoring (clicking a letter scrolls to that section)
13. Verify definition list grouped by letter with clickable titles

**Letter page (e.g., /dictionnaire/lettre/b/):**
14. Verify breadcrumbs: Dictionnaire > Lettre B
15. Verify definition list with category badges on each item

**Definition page (e.g., /dictionnaire/acinq/):**
16. Verify breadcrumbs: Dictionnaire > Organisation > ACINQ
17. Verify category badge "ORGANISATION" links to category page
18. Verify search bar present at top
19. Verify cross-reference pills (click one to navigate)
20. Verify prev/next navigation within category
21. Verify book CTA at bottom (Bitcoin Bazar + Amazon + GitHub buttons)
22. Navigate to a definition with images and click an image — verify lightbox opens
23. Press Escape — verify lightbox closes

**Mobile (resize to 375px or use DevTools):**
24. Verify category grid shows 2 columns
25. Verify CTA buttons stack to full width
26. Verify search dropdown is usable on mobile

```css
/* Lien skip-to-content (accessibilite) */

  left: -9999px;
  top: 0;

  color: var(--bg);
  font-family: var(--font-body);

  border-radius: 0 0 var(--radius) 0;

.skip-to-content:focus {
  left: 0;
}
```

**Step 2: Add missing focus states**

This is the most complex step. For each duplicated breakpoint:

1. Collect ALL rules from every `@media (max-width: 719px)` block (10 blocks scattered across the file)

3. Remove the now-empty original blocks
4. Repeat for `@media (max-width: 380px)` (2 blocks) and `@media (min-width: 720px)` (2 blocks)

Target media query block order at end of file:
1. `@media (max-width: 380px)` — 1 block
2. `@media (min-width: 720px)` — 1 block
3. `@media (max-width: 719px)` — 1 block (merged from 10)
4. `@media (min-width: 720px) and (max-width: 979px)` — 1 block

7. `@media (prefers-reduced-motion: reduce)` — 1 block (always last)

**Rules for consolidation (per D-03):**
- DO NOT restructure the entire file — only move media query contents

- Remove orphan/empty media query blocks after merging
- Do NOT change any actual CSS property values
- Do NOT add or remove any CSS rules (except skip-to-content and focus states above)

Dans le cadre du portefeuille Samourai Wallet et d'autres logiciels de portefeuilles qui l'implémentent, un Cahoot désigne tous les types de transactions réalisées en collaboration entre plusieurs utilisateurs. Procéder à un Cahoot signifie donc participer conjointement à une transaction. Cette collaboration s'articule autour de l'échange de transactions partiellement signées. Ces échanges peuvent se faire soit manuellement, via des codes QR, soit de manière automatisée, via le réseau de communication *Soroban*. Parmi les cahoots, on retrouve :
* Les transactions *Stowaway* (*Payjoin*) ;
* Les transactions *Stonewall x2* ;
* Les transactions *Joinbot*.
